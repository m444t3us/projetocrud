# projetocrud feito durante o IFSul

Este projeto foi desenvolvido como trabalho final da disciplina de Programação Web no IFSul.
Consiste em um sistema web com funcionalidades completas de **CRUD (Create, Read, Update, Delete)**, permitindo a manipulação de dados por meio de formulários integrados a um banco de dados.
O objetivo foi aplicar, de forma prática, as tecnologias aprendidas durante o curso, integrando frontend e backend em uma aplicação funcional.

# Linguagens utilizadas

- HTML5
- CSS3
- JavaScript
- php
- SQL (MySQL)

# Funcionalidades

- Cadastro de registros
- Listagem dos dados armazenados
- Atualização dos dados
- Exclusão de registros
- Inclusão ao banco de dados
